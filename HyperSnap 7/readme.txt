HyperSnap ver. 7

HyperSnap 7 is a screen capture and image editing
tool for Windows. Captures screens from standard
desktop programs and hard-to-grab DirectX/Direct3D,
mode games. It provides also powerful TextSnap(tm)
ASCII, UNICODE and RTF text capture functionality
from almost anywhere on the screen

HyperSnap 7 is distributed on "try before you buy"
principle.  If used without a license beyod the 20
days free trial period, it will add Hyperionics
stamps to images that you capture or edit with it.
To purchase a license, visit our web site at 
http://www.hyperionics.com
