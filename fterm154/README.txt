
kxn版fterm 2004.7 毕业纪念 之 fuse支援版

本版本为在kxn 2004.7毕业纪念版基础上，作的修正与改进。

http://fterm.zj001.net


o 图片预览 
     支持png 
     支持tiff 
     支持psp 
     支持动态gif 
     还有很多很多......
o xml地址簿 
     本软件解压后，将原来的fterm.str文件拷贝到此目录下，运行convert.exe，
     点击“ 转换”，会生成fterm.xml。 
     关闭convert.exe程序，再打开fterm.exe就有了原来地址簿中的所有条目。 
     采用xml地址簿后，将来升级时，此fterm.xml文件将可以复用
o Z-Modem下载文章按键
     地址项属性BBS设置中有一项“Z-Modem下载文章按键”，意指在支持Z-Modem的
     BBS站h 按什么键下载文章。 
     smth和ytht都是“^yY^M^M^M” 
     这个功能以前曾经发布过，几次维护中不知何时丢掉了
o Many bug fixed 
     这也意味着可能会有更多的新bug......

下面是 2004 年 7 月以来 kxn 版本的 fterm 相对于 2.5.0.122 官方版本的修改

20040722修正：

1: 支持图片 URL 的重定向
2: PicRules.ini 里面增加了武汉华中理工的图片链接正则。

毕业离校纪念版本：

1: 提供英文界面，程序运行中可以动态切换，语言文件为文本文件可自己编辑
   （非简体中文 OS 用户升级时请阅读里面的 升级须知.txt）
2: 重写了图片下载引擎，比原来的效率更高，应该能够解决很多系统的不能下载问题
   （很多人报 unsupported 其实是没有下载成功的错误显示，新版本中可以正确显示
    Error）。下载引擎使用 bbs 站点设定的代理，注意不支持 socks4 类型代理。
3: 支持除 smth 和一般图片 URL 之外的自定义格式 URL 预览，在 PicRules.ini 
   里面设置正则表达式，按照RegEx0,RegEx1,...这样排列，里面提供了一个北方
   交通大学红果园的例子。注意，所有英文字母都要变为小写。
4: 可以识别 ytht 文章链接中带有 * 和 + 的 URL，ytht 的图片预览仍然不能支持
5: 在 URL 链接上面点右键出来的菜单可以复制链接和打开链接，类似浏览器的风格。
6: 修正了预览图片窗口保存图片无效的 bug
7: 修正了最小化到tray后退出报错的 bug
8: 过滤掉 IP 提示中的 cz88.net 
9: 修正了 alt-f4 关闭预览窗口时候导致主窗口修正的 bug
10: 修正了窗口不是最大化状况下的查看IP窗口位置错误 bug
11: 修正了会错误将 ftp://1.1.1.1 这样的 URL 显示为 IP 的问题
12: 修正了默认地址本中浙江大学海纳百川 bbs 和 笑书亭的地址

Private Build 2.5.0.129

1：增加了看 IP 位置的功能，类似 QQ 里面的，鼠标放到 IP 上面会浮动显示
2: 现在只有鼠标移动到图片链接上之后才会开始下载图片，以前是只要屏幕中出现就开始
下载。
3：增加了一个设置选项，可以控制是否预览图片，默认为开，实在不能用的可以关上。
4: 修正了 128 里面引回的初次下载出错 bug

Private Build 2.5.0.127

1: 支持除 ytht 系统之外 bbs 的图片预览， ytht 的图片由于没有原始链接因此比较难于
显示。目前正在解决中
2：改善了一点点刷新速度，不过一般人可能感觉不出来。

Private Build 2.5.0.126

1: 最小化到 icon 的时候，浮动提示带窗口标题
2: 修正了 ssh 模式下面如果在 socket 连接上到弹出认证窗口这段时间里面有按键会导致
 Packet size too big 或者不停吃内存失去响应的 bug
3: 发现前几个  private build  fterm 主窗口图标坏掉了，修正了
4: 支持 .jpeg 扩展名的图片

Private Build 2.5.0.125

1: 修正了zmodem下载全文时候第一次报错的问题
2: 设置界面里面 砖换 -> 转换
3: 现在 fterm 启动时候会删除 pool 目录里面的所有文件，防止这个目录过大
4: 关机时候 fterm 不再询问是否断开（未经测试）

Private Build 2.5.0.124

1: SMTH 图片预览功能（由 FreeWizard@SMTH 和 Youngcow@SMTH 完成 )
   当鼠标光标移动到文章中的附件链接时，fterm会在预览窗口中显示图片。特别适用于
校内网络速度快的情况。

2: 输入法光标跟随功能 （由 kxn@SMTH 完成 )
   在 fterm 里面使用中文输入法输入文字的时候，输入法窗口会跟随光标移动，就像在
一般的文本编辑器中那样，特别在使用 MSPY2003 的时候，输入起来十分自然。
